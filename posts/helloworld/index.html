Hello world！

你好，互联网。

说来尴尬

域名？ DigitalPlat Domain 免费的。
托管？ GitHub Pages 不要钱。
主题？ GitHub 上 fork 的，感谢那位不知名大佬。

完美闭环。零成本开站。

为什么折腾这个

我想了想，这时的我，似乎没什么慰藉。

但至少——
在浏览器地址栏敲一串字符，
就能敲开一个完全属于我自己的页面。
没有广告，没有审核，
没有“因违规内容无法查看”。

helloworld 是给谁看的

理论上，是给全世界看的。

实际上，大概只有我自己，还有偶尔爬虫路过。

没关系。我把这个页面当成一个数字基地。以后所有白嫖来的小玩意、自己写的烂代码、折腾三天三夜终于跑通的教程，都堆在这儿。

等我老了（虽然现在才14），回头看这个暑假，除了补考数学，至少还干成过一件事。

希望一直走下去

其实不太敢说“一直”。

我的持久度记录：我的世界存档玩了4个月，然后格式化硬盘，没备份，于是没了；洋葱学院APP打卡最长68天，然后开学了忘了买保持记录的玩意；这个博客……

不知道。先活过初三下学期再说。

但只要 GitHub 不倒闭，DigitalPlat Domain 不收回域名，这个页面就会一直挂在那里。哪怕我半年不更新，哪怕我跑去干别的事了。
它像一个漂流瓶，漂在2025年的夏天。

保存，生成，部署，上传。

等待1分钟，郑重地输入那串域名。

从零到一，从空白到有字，从“我想”到“我做了”。

花了整个下午，成本零元，成就感爆棚。

这个下午值得记住。不是因为建成了博客，是因为终于有件事，我让自己坚持到了“完成”。

不保证能写多久。

但至少，今天写了。

这一行，就是开始。
